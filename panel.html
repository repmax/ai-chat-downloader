Welcome to Chat Downloader!

This browser extension allows you to capture and download entire chat sessions from **perplexity.ai**, **you.com** and **claude.ai** in markdown format. AI chat bot services offer only to grab a single response at a time, this tool grabs the full chat directly from the providers servers.

Key Features:

  • Retrieve full chat sessions in markdown format, including frontmatter.
  • Customize tags, date, title, or any other properties before downloading.
  • Modify the filename to your preference.

Usage Instructions:

  1. Ensure your current tab is a chat session on perplexity.ai, you.com or claude.ai.
  2. Click the "Generate" button. A reload of the page will be triggered to fetch the latest chat data.
  3. Edit the markdown content and frontmatter as needed.
  4. Change the filename if desired.
  5. Click "Download" to save the markdown file.

This extension is ideal for users leveraging advanced knowledge management systems like Obsidian. Archive longer conversations and compare responses from different language models with ease.

Enjoy using Chat Downloader!
